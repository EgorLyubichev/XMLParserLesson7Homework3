1. Забрать проект XMLParser с github
создать объект этого класса, передав ему путь к файлу с XML: new XMLParser(path).
Затем вызвать метод parseDocument. Наличие ошибок можно проверить, вызвав после этого у объекта метод getErrors.
Если список пуст, значит xml признан валидным. Фокус должен быть на наполнение разнообразными входными xml файлами и поиск багов
в приложении.
3. Следующие возможности XML не поддерживаются парсером согласно требованиям:
подключение схем
пространства имен
doctype
любая иная неработающая функциональность является багом.
